Lista dos Melhores Sites que já vi

Top 3 dos melhores sites que existem na web

  1. Google
  2. Trybe
  3. Pinterest

Principais motivos para eu gostar deles

Google

  • Aprender coisas novas;
  • Fazer consultas para qualquer tipo de dúvida que possa surgir;
  • Ajuda para organizar muitas informações utilizadas no dia-a-dia com ferramentas ideais.

Trybe

  • Consulta livre para recaptular todos os conteúdos que precisar ao longo do dia;
  • Muitos assuntos adicionais que ajudam além da programação, como Linkedin, VSCode e Soft Skills;
  • Recursos adicionais, para aprofundar os novos conhecimentos e ter um grande aproveitamento das informações que eles nos oferecem.

Pinterest

  • Todo tipo de ideias com muita criatividade;
  • Possibilidade de criar pastas com todos os interesses que queremos colecionar;
  • Acessar interesses antigos sempre que quiser.
